#Git spurningar og svör
1. Hvað gera eftirfarandi Linux skipanir?
	cd: breytir um efnisskrá
	cd ~ : breytir í heimasvæði
	ls: sýnir innihald efnisskrárnar sem þú ert að nota
	pwd: sýnir hvar efnisskráinn sem þú ert að nota er
	mkdir: býr til nýa efnisskrá

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone: afritar geymslu í nýa efnisskrá, getur líka verið notað til að endurnefna skjöl
	git log: sýnir skráningu á aðgerðum sem hafa viðbæt efnisskráni sem á við, gott til að vista í mismunandi gerðir að sama hlutnum
	git status: skilgreinir hvaða skrár eru "staged", "unstaged" og "untracked"
	git diff: finnur munin milli tveggja hluta eins skrám commits eða branch, best er að nota það til að sjá hverju hefur verið bætt á skjali á mismunandi tímum
	git checkout: notað til að skoða eldri gerðir að hlutnum sem þú ert að nota án þess að yfirita það sem þú ert að gera

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff : framkvæmir skipunina á venjulegan hátt eins og lýst er fyrir ofan
	b) git diff --staged :það sýnir munin á HEAD skjalinu miðað við það sem þú valdir að nota í skipunini
	c) git diff commit1 commit2: þessi skipun mun ekki virka án einhverra skala, commits osf. sem heita "commit1" og "commit2"

4. 	Hvað er útgáfustýring (e.version control)? : það er bara létt orð til að lýsa forritum sem vinna með mismunandi útgáfur af sama hlutnum
